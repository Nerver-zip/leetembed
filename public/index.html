LeetEmbed

How to Use

To generate a Discord preview for a LeetCode question, append the question slug to the URL:

https://leetembed.vercel.app/api/problems/two-sum

Example:

https://leetembed.vercel.app/api/problems/two-sum

Details

  • Discord will automatically generate a preview using the OG meta tags.
  • The description is cleaned to show only the problem statement without examples.
  • You can copy any LeetCode question slug and paste it in the URL.

Default Image

The preview image will always use the LeetCode logo unless the OG image is changed.
